1. Как сдавать алгоритмические задачи ?
Переходите по ссылке на задание
Нажимаете на кнопку Fork справа вверху
Копия репозитория появляется в вашем аккаунте
Скачиваете на компьютер собственную копию репозитория git clone ссылка-на-репозиторий
Заходите в папку репозитория. Запускаете команду npm install Ждёте
Когда команда выполнилась, Заходите в папку src, открываете файл index.js. Пишете код, решаете таск.
Чтобы проверить правильность выполнения задания, выполняете команду npm test
Если проверка пройдена, заливаете изменения на гитхаб. Выполняете команды git add .
git commit -m "feat: add task"

git push origin master

Заходите на страницу https://app.rs.school/course/submit-task?course=rs-2020-q1 Добавляете свой таск
